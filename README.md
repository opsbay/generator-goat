## Getting Started

### Install
To install generator-goat from npm, run:

```bash
npm install -g generator-goat
```

Finally, initiate the generator:

```bash
yo goat
```

### CI with Wercker
Register with [Wercker](http://wercker.com/) and create your app (Wercker's guides are quite clear).
